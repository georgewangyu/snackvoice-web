# SnackVoice Frontend Tokens

## Direction

- Mood: dark, quiet, technical, premium
- Composition: poster-like hero with one dominant transcript visual
- Voice: product utility, not generic SaaS hype

## Core Tokens

- Background: `#171512`
- Deep background: `#100f0d`
- Text: `#f6f3ee`
- Muted text: `rgba(246, 243, 238, 0.72)`
- Accent: `#8a7cff`
- Accent deep: `#6f61ef`
- Success accent: `#8be6bb`

## Typography

- UI font: `Inter`
- Technical / supporting font: `IBM Plex Mono`
- Hero headline: `clamp(3.7rem, 8vw, 7rem)`
- Section title: `clamp(2.25rem, 4vw, 4rem)`

## Layout

- Site container: `min(1180px, calc(100% - 48px))`
- Hero layout: two-column with a full-bleed feel and a dominant visual shell
- Radius system: `10px / 18px / 28px / 40px`

## Motion

- Reveal-on-scroll for section entrances
- Subtle pulsing waveform in hero
- Rotating transcript lines in the main visual
